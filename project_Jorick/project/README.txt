Final assingment


Description
Instructions to run the simulation for the assingment. The final.rmd contains the markdown file of the assingment.
The file final.pdf contains the pdf view of the task.


Installation
Conditions to run the simulation:
- R version 4.0.0 or higher
- Download the r package 'deSolve' version 1.28 or higher
- for kniting the .rmd file. Make sure the references.bib file is in the same directory. 


Usage
After the installation requirements it sould run and generate the pdf without any problems.
